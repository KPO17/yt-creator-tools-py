Important : Cette politique explique comment nous traitons vos données lors de l'utilisation de YT Creator Tools.

1. Données collectées

Nous ne collectons que les données minimales nécessaires :

  • Liens YouTube temporairement (pour traitement)
  • Email (si vous créez un compte)
  • Données d'usage anonymes

2. Utilisation des données

Vos données sont utilisées uniquement pour :

  • Fournir le service d'extraction de métadonnées
  • Gérer votre compte utilisateur
  • Améliorer l'application

3. Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers, sauf :

  • Services techniques nécessaires (hébergement)
  • Obligations légales

4. Vos droits

Vous avez le droit de :

  • Accéder à vos données
  • Les corriger ou les supprimer
  • Retirer votre consentement

5. Contact

Pour toute question : contact@ytcreatortools.com

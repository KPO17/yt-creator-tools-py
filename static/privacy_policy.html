Politique de Confidentialité

YT Creator Tools

Important : Cette politique explique comment nous traitons vos données lors de l'utilisation de YT Creator Tools.

1. Données collectées

YT Creator Tools collecte uniquement les données minimales nécessaires au fonctionnement :

  • URLs YouTube : Traitées temporairement pour extraire les métadonnées (non stockées)
  • Données d'usage anonymes : Statistiques générales d'utilisation pour améliorer le service
  • Logs serveur : Fichiers journaux techniques pour la maintenance

2. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Fournir le service d'extraction de métadonnées YouTube
  • Générer et télécharger les miniatures
  • Extraire les sous-titres des vidéos
  • Améliorer les performances de l'application
  • Résoudre les problèmes techniques

3. Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers, sauf :

  • Services techniques nécessaires : Hébergement (Render.com) pour le fonctionnement de l'application
  • Obligations légales : Si requis par la loi ou une décision de justice
  • API YouTube : Les requêtes sont transmises à YouTube selon leurs conditions d'utilisation

4. Sécurité des données

Nous mettons en place des mesures de sécurité appropriées :

  • Connexions HTTPS chiffrées
  • Traitement temporaire des URLs (non stockées)
  • Accès limité aux données techniques
  • Surveillance des accès non autorisés

5. Conservation des données

  • URLs YouTube : Traitées immédiatement puis supprimées
  • Fichiers téléchargés : Générés à la demande, non stockés sur nos serveurs
  • Logs techniques : Conservés maximum 30 jours
  • Données analytiques : Anonymisées et conservées pour amélioration du service

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Connaître les données que nous traitons
  • Droit de rectification : Corriger les données inexactes
  • Droit à l'effacement : Demander la suppression de vos données
  • Droit d'opposition : Vous opposer au traitement de vos données
  • Droit à la portabilité : Récupérer vos données dans un format standard

7. Cookies et technologies similaires

L'application utilise :

  • Stockage local : Pour sauvegarder vos préférences d'utilisation
  • Service Worker : Pour le fonctionnement hors ligne (PWA)
  • Pas de cookies de tracking : Aucun suivi publicitaire

8. Services tiers

L'application utilise les services suivants :

  • YouTube API : Pour récupérer les métadonnées (soumis aux conditions YouTube)
  • Render.com : Hébergement de l'application
  • YouTube Transcript API : Pour l'extraction des sous-titres

9. Modifications de cette politique

Cette politique peut être mise à jour pour refléter :

  • Les changements dans nos pratiques
  • L'évolution de la réglementation
  • L'ajout de nouvelles fonctionnalités

Les modifications importantes seront communiquées via l'application.

10. Contact

Pour toute question concernant cette politique ou vos données :

  • Email : privacy@ytcreatortools.com
  • Réponse : Sous 72 heures maximum
  • Délégué à la protection des données : dpo@ytcreatortools.com
Note importante : YT Creator Tools fonctionne en mode "privacy-first" - nous ne stockons aucune donnée personnelle et traitons vos requêtes de manière temporaire uniquement.
